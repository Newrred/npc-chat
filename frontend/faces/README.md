# Face Asset Naming

Put base portraits in this folder for GitHub Pages deployment.

Default rule:
- URL template: `./faces/{face_slug}.png`
- `face_slug` is lowercase with spaces replaced by `_`

Required file names:
- `neutral.png`
- `happy.png`
- `sad.png`
- `angry.png`
- `crying.png`
- `smiling.png`
- `smirk.png`
- `shy_smile.png`
- `blushing.png`
- `teary.png`
- `surprised.png`
- `confused.png`
- `annoyed.png`
- `pouting.png`
- `tired.png`
- `scared.png`
- `excited.png`
